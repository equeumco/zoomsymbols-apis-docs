## Set Profile

Set your user profile

```
POST https://dev.zoomsymbols.com/api/user/setProfile
```
> Params

```json
{
  "first_name": "Alex",
  "last_name": "Alex",
  "phone_number" : "xxxxx",
  "title" : "Developer",
  "bio": "Your bio"
}
```
